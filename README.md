# Digital-Banking-QA-Suite
A QA Automation project focused on testing real-world digital banking workflows such as user authentication, fund transfers, transaction validation, and account management. This repository demonstrates manual testing, defect reporting, Selenium-based automation, API validation, and Agile QA practices using a simulated banking application.

# Digital Banking QA Automation Suite

## Project Overview
A QA Automation project focused on testing digital banking workflows such as user authentication and fund transfers using Selenium and Python. The project demonstrates manual testing, negative testing, and automation testing practices on a simulated banking application.

## Application Under Test
Parabank Demo Banking Application:
https://parabank.parasoft.com/parabank/index.htm

## Tools & Technologies Used
- Python
- Selenium WebDriver
- PyTest
- VS Code
- Git & GitHub

## Test Scenarios Covered

### Login Testing
- Valid Login
- Invalid Login
- Empty Username/Password Validation

### Fund Transfer Testing
- Successful Fund Transfer
- Invalid Transfer Amount
- Empty Amount Validation
